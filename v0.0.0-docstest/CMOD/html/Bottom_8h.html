Detailed Description

Leaf event in a DISSCO composition.

Unlike higher layers, a Bottom event does not spawn further sub-events: it is the point where the composition tree turns into renderable output. Each Bottom emits a sequence of Sounds (for synthesis) and/or Notes (for traditional notation), driven by frequency / loudness / modifier elements that are re-evaluated per child to allow stochastic variation.
