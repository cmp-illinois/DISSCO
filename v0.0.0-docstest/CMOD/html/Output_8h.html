Detailed Description

Side-band result tree CMOD builds while expanding a Piece.

As events resolve into sounds and notes, the Output static class accumulates an OutputNode tree that mirrors the event hierarchy. Each node carries (name, value, unit) properties and a list of child nodes; the tree is serialized to the project's .particel XML report when synthesis finishes.
