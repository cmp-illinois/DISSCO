Detailed Description

Core Event class — the common base for every layer in a DISSCO composition tree (top, high, mid, low, bottom).

Each Event holds the metadata that is shared regardless of layer: timespan, tempo, pattern cache, the XML nodes for the sieve / spa / reverb / filter / modifier attribute blocks, and the bookkeeping needed to merge ancestor modifiers when expanding children. Layer-specific behaviour lives in subclasses (e.g. Bottom).
