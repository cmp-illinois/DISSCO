Detailed Description

Central include hub for CMOD.

Pulls in the C++ standard library headers CMOD touches, the LASS audio library (LASS.h), the templated rational arithmetic (Rational.h), pugixml for project parsing, and muParser for arithmetic-expression evaluation. Also declares the EventType, TempoPrefix, and TempoNoteValue enums plus the small pugixml helpers (GFEC, GNES, descendantByName) that emulate the Xerces "element-only" traversal CMOD originally targeted, and applies the project-wide using namespace std.
