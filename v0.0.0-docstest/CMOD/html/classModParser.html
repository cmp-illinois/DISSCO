Detailed Description

Builds an integer set from a sieve expression string.

Holds the parser's working state: the modulus list, the offset list, and the resolved element set after parsing. The private Token helper lets operators be polymorphic across numbers and lists so future operations (e.g. offsetting a list by an integer) drop in cleanly.
