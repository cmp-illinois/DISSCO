Detailed Description

3-D event-selection probability matrix.

Builds an (event type × start time × duration) matrix of occurrence probabilities, normalizes it, samples one event from it, and re-balances the matrix after each choice so subsequent picks reflect what has already been used. Originally written by Ryan Cavis.
