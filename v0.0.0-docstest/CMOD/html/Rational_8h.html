Detailed Description

Templated exact-rational arithmetic, adapted from Belle, Bonne, Sage (William Andrew Burnson, 2007–2010).

CMOD's Tempo and EDU bookkeeping are expressed as rationals to avoid floating-point drift across long pieces. The whole class is header-only because it is templated on the underlying integer type; Ratio (an int specialization) is the alias the rest of CMOD uses.
