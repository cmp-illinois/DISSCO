Detailed Description

Decay. Three numeric entries (Base/Rate/Index) plus a Type radio (Exponential/Linear) that sits between Base and Rate in the XML.

<Fun><Name>Decay</Name> <Base>V</Base> <Type>EXPONENTIAL|LINEAR</Type> <Rate>V</Rate> <Index>V</Index> </Fun>
