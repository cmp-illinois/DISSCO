Detailed Description

MakeFilter: filter-type radio (7 options) plus Frequency / BandWidth / dBGain entries. dBGain is only meaningful for HSF / LSF / PBEQF filters; for the others the row is disabled.

<Fun><Name>MakeFilter</Name> <Type>LPF|HPF|BPF|HSF|LSF|NF|PBEQF</Type> <Frequency>V</Frequency> <BandWidth>V</BandWidth> <dBGain>V</dBGain> </Fun>
