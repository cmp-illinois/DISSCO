Detailed Description

Aggregate state for a single open DISSCO project.

All members are private; ProjectManager is the only friend and is the intended access path for the rest of LASSIE. Construction is restricted — client code goes through ProjectManager::create() / ProjectManager::open() rather than instantiating Project directly.
