Detailed Description

In-app "Check for Updates" client backed by the GitHub Releases API.

GETs /releases/latest, parses the tag as a QVersionNumber, and compares it against the build's compile-time DISSCO_VERSION. If newer, offers to download the matching platform asset (DMG / AppImage / EXE) and hands it to the OS to install.

Triggered manually from the Help menu, and optionally on app startup once per 24h.
