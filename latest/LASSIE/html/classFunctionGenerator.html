Detailed Description

Dialog that hosts one FunctionWidget at a time, selected via a combo box filtered by the requested return type. The dialog is now just a thin shell – it pulls the available functions from FunctionRegistry, instantiates them on demand, and mirrors each widget's xmlChanged signal into the result text edit. All per-function UI and slot logic lives in the FunctionWidget subclasses under dialogs/functions/impl/.
