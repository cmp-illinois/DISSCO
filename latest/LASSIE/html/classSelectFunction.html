Detailed Description

Select: a list of values (each a "Select" row) plus an Index field that picks one of them.

<Fun><Name>Select</Name>
<List>v1, v2, <Fun>...</Fun>, ...</List>
<Index>V</Index>
</Fun>

Items in <List> are joined with ", "; nested <Fun>...</Fun> values are preserved verbatim and the parser splits only on top-level commas.
