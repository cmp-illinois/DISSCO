Detailed Description

MakeEnvelope: a dynamic list of (X, Y, interpolation, proportion) points plus a Scale factor.

<Fun><Name>MakeEnvelope</Name> <Xs><X>v</X>...</Xs> // N values <Ys><Y>v</Y>...</Ys> // N values <Types><T>LINEAR|SPLINE|EXPONENTIAL</T>...</Types> // N-1 (segments) <Pros>

FLEXIBLE|FIXED

...</Pros> // N-1 (segments) <Scale>v</Scale> </Fun>
