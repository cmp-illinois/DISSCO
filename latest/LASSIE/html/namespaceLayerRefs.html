using LayerRefs::Assoc = typedef QList<QPair<OwnerKey, QList<int> >>

Result of a project-wide reference scan.

Each entry pairs an OwnerKey with the indices, within that layer's discrete_packages list, of the Packages that matched the (type, name) the scan was looking for.
