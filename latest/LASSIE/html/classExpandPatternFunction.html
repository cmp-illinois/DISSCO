Detailed Description

ExpandPattern: method radio + four entries (Modulo/Low/High/Pattern).

<Fun><Name>ExpandPattern</Name>
<Method>EQUIVALENCE|SYMMETRIES|DISTORT</Method>
<Modulo>V</Modulo>
<Low>V</Low>
<High>V</High>
<Pattern>V</Pattern>
</Fun>
