Detailed Description

Envelope reinterpreted as a probability density / cumulative distribution.

Authored by Fanbo Xiang. Reused by CMOD bottom events that need to sample a value whose distribution is itself an envelope shape — the area under the curve becomes the total mass, and inverse-CDF sampling picks an x-coordinate.
