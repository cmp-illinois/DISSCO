Detailed Description

This program implements the biquad transfer function b0 + b1*z^-1 + b2*z^-2 H(z) = ---------------------—
a0 + a1*z^-1 + a2*z^-2 according to the specs at http://www.musicdsp.org/files/Audio-EQ-Cookbook.txt. low pass filter, High pass filter,band pass filter , Notch Filter , Peaking band EQ filter, Low shelf filter, High shelf filters are implemented author Mert Bay
