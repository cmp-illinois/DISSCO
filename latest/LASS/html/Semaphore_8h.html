Detailed Description

Counting semaphore shim over std::mutex / std::condition_variable.

C++11 has no std::counting_semaphore (that arrived in C++20), so this supplies the POSIX sem_init / wait / post semantics LASS uses to throttle the per-sound rendering workers in Score, without pulling in pthreads. Non-copyable to prevent the implicit move that would invalidate the embedded mutex.
