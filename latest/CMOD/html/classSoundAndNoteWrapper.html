Detailed Description

Per-child snapshot used while a Bottom event is building its notes.

Bundles the XML element describing the child (either a spectrum entry, if the bottom is a Sound, or a note entry), the resolved timespan, the resolved tempo, and the child's display name / numeric type so the Bottom's expansion routines can iterate without going back to the parser.
