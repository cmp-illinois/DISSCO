Detailed Description

Pattern generator that emits one value per call.

Authored by Sever Tipei (July 2005, revised July 2007). Builds a pattern from a user-supplied origin plus a sequence of intervals, then yields values one at a time. The same input can be transformed into related patterns via mod-m equivalences, symmetrical set forms (R, I, RI), or intentional distortions. PAT files on disk describe both the object and the pattern that drives it.
