Detailed Description

CMOD command-line entry point.

Parses arguments, installs signal handlers, instantiates the Piece model from the supplied .dissco project file, and drives it through parse, expand, and emit-output. All long-lived state is owned by the Piece — Main.cpp is a thin orchestrator.
