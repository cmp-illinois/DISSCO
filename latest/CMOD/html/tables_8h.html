Detailed Description

Static lookup tables used by the notated-score path.

Pitch names (sharps and flats), tuplet name labels, rhythmic-ratio names, rest-sign labels, the canonical list of note-modifier strings, and a few power-of-two lookups. The arrays are declared extern here and defined once in tables.cpp.
