Detailed Description

Notated-score model rendered out as LilyPond.

Original notation module written by Haorong Sun; multistaff extensions tagged "multistaffs" in the source were added by Xiaoyi Han. A NotationScore owns the per-staff stream of Section objects, the tempo / time-signature spine, and the metadata (title, staff count, grand-staff layout) that LilyPond needs to typeset the result.
