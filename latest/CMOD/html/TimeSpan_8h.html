Detailed Description

Dual-representation [start, end] interval used by every Event.

Each event lives on two clocks at once: the audible / wall-clock seconds used by LASS for synthesis, and the rational EDU grid used by CMOD for notated rhythm. TimeSpan carries both, plus the absolute EDU offset needed when stitching multiple events onto a single global timeline.
