Detailed Description

Project-wide tuning constants and audible-range limits for CMOD.

Holds the small set of physical and musical constants the engine references when interpreting a piece: amplitude ceiling, minimum audible frequency, the equal-temperament step ratio, C0 in hertz, and similar tuning bases. Pure constants — no functions, no state.
