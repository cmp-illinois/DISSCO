Detailed Description

Concrete leaf Event that produces Sounds and Notes.

Bottom keeps the XML element handles for the per-event attribute blocks (frequency, loudness, modifiers, modifier group, ancestor modifiers) and recomputes them for every child it emits — the same Bottom can produce different frequencies / loudnesses on successive children when the driving expressions contain randomness.

Bottom also tracks the bookkeeping needed for notated output (current partial, well-tempered pitch number, absolute pitch number, modifier usage map) and owns the list of child Sounds.
