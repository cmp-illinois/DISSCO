Detailed Description

Expression parser for modular-arithmetic sieve definitions.

Parses the small infix language used by sieve attributes — modular residue classes (m mod n), unions / intersections / differences / complements over those sets, and integer offsets — into a flat list of allowed integers. The supported operators have hard-coded precedence so no grammar table is needed.
