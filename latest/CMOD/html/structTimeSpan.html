Detailed Description

Interval expressed simultaneously in seconds and in EDUs.

start / duration / end are seconds; startEDU / durationEDU / endEDU are exact rationals on the EDU grid. startEDUAbsolute is the cumulative integer EDU offset from the start of the piece.
