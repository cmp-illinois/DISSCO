Detailed Description

Custom POSIX signal handlers for CMOD (Rubin Du, 2024).

Right now only segfaultHandler is wired up — it prints a symbolized stack trace before exiting, so a crash mid-render produces something actionable in the terminal log. The interrupt / terminate / abort hooks are declared but intentionally left empty; they exist as anchor points for cleanup logic once a graceful-shutdown story is needed.

BACKTRACE_NUM bounds the number of frames captured.
